# Theming Superset

:::note
apache-superset>=6.0
:::

Superset now rides on **Ant Design v5’s token-based theming**.
Every Antd token works, plus a handful of Superset-specific ones for charts and dashboard chrome.

## 1 — Create a theme

1. Open the official [Ant Design Theme Editor](https://ant.design/theme-editor)
2. Design your palette, typography, and component overrides.
3. Open the `CONFIG` modal and paste the JSON.

You can also extend with Superset-specific tokens (documented in the default theme object) before you import.

## 2 — Apply it instance-wide

You can configure themes through multiple methods:

### Method 1: Configuration File

```python
# superset_config.py
THEME_DEFAULT = {
    # Paste your JSON theme definition here
}

# Optional: Configure dark theme
THEME_DARK = {
    # Dark theme configuration
}

# Optional: Theme behavior settings
THEME_SETTINGS = {
    "enforced": False,        # Don't force the theme
    "allowSwitching": True,   # Allow users to switch themes
    "defaultTheme": "light",  # Default theme selection
    "rememberUserPreference": True  # Remember user's theme choice
}
```

### Method 2: Environment Variables

```bash
# Set default theme via environment variable
export SUPERSET__THEME_DEFAULT='{"token":{"colorPrimary":"#1890ff"}}'

# Set dark theme
export SUPERSET__THEME_DARK='{"token":{"colorPrimary":"#177ddc"}}'

# Configure theme behavior
export SUPERSET__THEME_SETTINGS='{"allowSwitching":true,"defaultTheme":"light"}'

# Or use nested environment variables
export SUPERSET__THEME_DEFAULT__token__colorPrimary="#1890ff"
export SUPERSET__THEME_SETTINGS__allowSwitching=true
```

### Method 3: Database Configuration (Coming Soon)

Future versions will support runtime theme configuration through the admin interface.

**Note:** File-based and environment variable changes require a restart to take effect.

## 3 — Tweak live in the app (beta)

Set the feature flag in your `superset_config`
```python
DEFAULT_FEATURE_FLAGS: dict[str, bool] = {
    {{ ... }}
    THEME_ALLOW_THEME_EDITOR_BETA = True,
}
```

- Enables a JSON editor panel inside Superset as a new icon in the navbar
- Intended for testing/design and rapid in-context iteration
- End-user theme switching & preferences coming later

## Verify Theme Configuration

You can inspect your current theme configuration using the CLI:

```bash
# View current theme settings
superset config get THEME_DEFAULT
superset config get THEME_DARK
superset config get THEME_SETTINGS

# Show all theme-related configuration
superset config show --filter "THEME"
```

This will show you the current values and their sources (environment variables, config file, or defaults).
